XOR Encryption
Encrypt and Decrypt Messages

This tool uses the bitwise exclusive or operator to encrypt data. A key of any size can be used, but a key of about the same length as the message will make it significantly harder to crack.

Type a plaintext message into the message textarea, and either supply or generate a key. Then hit Encrypt. The result will be given to you in base64 to prevent non-printable ASCII characters. If you would like to decrypt, just paste the base64 value into the message textarea with the same key and hit Decrypt.
